  • Frågor om dina produkter Om det i samband med att du läser din journal uppkommer medicinska frågor eller funderingar kan du få svar på olika sätt:
  • På www.1177.se hittar du fakta och praktiska råd om sjukdomar och besvär, undersökningar, behandlingar och läkemedel. Välj "Fakta och råd".
    Du kan även skicka en anonym skriftlig fråga via www.1177.se och få svar på denna inom en vecka. Välj "Ställ en anonym fråga".
    Vid behov av personlig kontakt går det bra att ringa telefonnummer 1177 "Råd om vård dygnet runt".

    Du kan också kontakta din vårdenhet genom att logga in på 1177.se och skicka meddelande till berörd mottagning eller via telefon, på vårdenhetens angivna telefontider.
