Hämta på utlämningsställe

Här väljer du utlämningsställe. Du kan även välja att någon annan/kontaktperson ska hämta ut paketet. Det vill säga en annan person än du själv har rättighet att hämta ut paketet.

Utlämningsställe

Ditt närmsta utlämningsställe baserat på ditt postnummer todo om postnummer inte kan hämtas Ändra postnummer

Obs. Valda artiklar kan inte levereras av en enskild transportör. Du behöver därför välja flera utlämningsställen.

Avisering

Avisering skickas per sms, e-post eller brev. Väljs sms måste mobilnummer anges, väljs epost måste e-postadress anges, väljs brev så skickas det till din folkbokföringsadress, detta går inte att ändra.

Avi:n skickas som (#{utilController.toProviderName(provider)})
Folkbokföringsadress:
#{userProfileController.userProfile.userProfile.firstName} #{userProfileController.userProfile.userProfile.lastName}
#{userProfileController.userProfile.userProfile.streetAddress}
#{userProfileController.userProfile.userProfile.zip} #{userProfileController.userProfile.userProfile.city}
